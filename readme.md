Pre-Entrega de Proyecto: Centro de Alto Rendimiento Martinez
Este proyecto es la primera entrega obligatoria del curso de Front End en Talento Tech. Se trata de una landing page responsiva diseñada para un centro de entrenamiento deportivo.

Requisitos Técnicos Cumplidos
Estructura Semántica
Se utilizaron etiquetas HTML5 como header, nav, main, section y footer para organizar el contenido de forma lógica.

Navegación Interna
Se implementó un menú con una lista desordenada y enlaces internos que permiten desplazarse por la página (Inicio, Planes, Reseñas y Contacto).

Estilos y Tipografía
Uso de Google Fonts (Roboto) y estilos CSS externos. Se aplicó un degradado (linear-gradient) en la sección principal para cumplir con el requisito de background.

Layout con Flexbox
La sección de Planes se organizó utilizando Flexbox en formato de tarjetas, permitiendo que los elementos se acomoden según el ancho de la pantalla.

Layout con CSS Grid
La sección de Reseñas utiliza CSS Grid para distribuir los testimonios de los atletas en una cuadrícula organizada.

Diseño Responsivo
Se aplicaron Media Queries para asegurar que el formulario de contacto y el resto de las secciones se adapten correctamente a dispositivos móviles.

Contenido Multimedia
Integración de un video mediante la etiqueta iframe para mostrar contenido audiovisual relacionado con el entrenamiento.
